# Python Socket Chat
A client-server chat application built with Python sockets and multithreading.

## What it does
- Server listens for incoming client connections
- Clients can send messages to the server in real time
- Supports multiple clients using threading

## What I learned
- Python socket programming
- Client-server architecture
- Multithreading
- Debugging network connection issues

## How to run
1. Start the server: 'python3 server.py'
2. Run the client: 'python3 client.py'
3. Type a message and press Enter to send

## Technologies
- Python 3
- Socket Library
- Threading Library
